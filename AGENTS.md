# AGENTS.md

This file provides guidance to AI coding agents when working with code in this repository.

## Project Overview

Chat is a natural language to zsh shell command translator for macOS. It uses Ollama with `qwen3-coder:latest` to convert English descriptions into shell commands, presented via an interactive TUI.

## Build and Run

```zsh
# Build
cargo build --release

# Run (launches TUI by default)
cargo run

# Lint (strict pedantic mode enabled via #![warn(clippy::all, clippy::pedantic)])
cargo clippy
```

**Prerequisites:** Ollama must be running (`ollama serve`) with the configured model installed.

## Configuration

The model can be configured via (in priority order):

1. `CHAT_MODEL` environment variable
2. Config file (`~/Library/Application Support/chat/config.toml` on macOS, `~/.config/chat/config.toml` on Linux):

   ```toml
   model = "qwen3-coder:latest"
   ```

3. Default: `qwen3-coder:latest`

On startup, the app verifies Ollama is running and the model is installed, showing a progress indicator while loading.

## Architecture

Single-file Rust application (`src/main.rs`) using:

- **clap** for CLI argument parsing with subcommand support (`tui` default, `run` for legacy mode)
- **ollama-rs** for LLM interaction with streaming responses
- **tokio** async runtime for I/O operations
- **color-eyre** for error handling
- **ratatui** + **crossterm** for TUI rendering and terminal control
- **arboard** for clipboard integration

### TUI Mode (default)

The application provides an interactive terminal interface with:

1. **Input box** - Type a natural language description of the desired command
2. **Loading state** - Spinner animation while waiting for LLM response
3. **Command selection** - Displays 1-3 command options generated by the LLM
4. **Explanation pane** - Shows explanation for the currently highlighted command
5. **Clipboard integration** - Selected command is copied to clipboard on exit

**Keyboard shortcuts:**

- `Enter` - Submit query / Select command / Select history item
- `↑/↓` - Navigate command options or history dropdown
- `←/→` - Move cursor in input
- `Home/End` - Jump to start/end of input
- `r` - Regenerate options with same query
- `x` - Delete selected history entry
- `Esc` - Cancel / Return to input / Exit
- `Ctrl+C` - Force exit

**History:**

- Previous queries are saved and shown in a dropdown while typing
- History is filtered as you type (case-insensitive)
- Stored in `~/Library/Caches/chat/history` (macOS) or `~/.cache/chat/history` (Linux)

### LLM Response Format

The TUI mode uses a structured prompt that requests 1-3 command options in the format:

```
1. command | explanation
2. command | explanation
3. command | explanation
```

The system prompt encourages use of modern CLI tools: jq, bat, eza, fd, fzf, rg, xh.
